# Space Tourism

A multi-page React application showcasing space tourism destinations, crew members, and technology.

## Tech Stack

- React
- CSS Modules
- Context API for state management

## Features

- Responsive design for mobile, tablet, and desktop
- Smooth page transitions with slide animations
- Dynamic background images per page
- Interactive navigation with active states

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in browser.

### Build for production

```bash
npm run build
```

## Project Structure

- `/src/components` - Reusable components (NavBar, PageContainer, etc.)
- `/src/pages` - Page components (Home, Destination, Crew, Technology)
- `/src/context` - Context providers for shared state
- `/src/assets` - Images and static assets
